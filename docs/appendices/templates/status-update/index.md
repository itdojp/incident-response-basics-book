# 状況共有テンプレ

## 更新頻度（目安）

| Severity | 更新頻度（目安） | 補足 |
|---|---|---|
| P0 | 15 分ごと | 影響/復旧見込みが変わらなくても「状況維持」を共有する |
| P1 | 30〜60 分ごと | 主要な切り分け結果・次アクションを更新する |
| P2 | 数時間ごと | 影響範囲/暫定対応/恒久対応の方針を更新する |
| P3 | 定期見直し（例: 週次） | バックログ管理し、放置ではなく管理する |

## 更新に含める最小項目

- 影響（誰に/何が/どの程度）
- 次アクション（担当/期限）
- 次回更新予定時刻（次にいつ共有するか）
